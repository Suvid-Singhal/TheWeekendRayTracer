# TheWeekendRayTracer
The ray tracer I built while learning graphics programming from the [Ray Tracing in One Weekend](https://raytracing.github.io/books/RayTracingInOneWeekend.html) by Dr. Peter Shirley.

This was more of an attempt to learn modern C++ fundamentals while exploring a really cool domain!
